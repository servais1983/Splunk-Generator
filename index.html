Getting Started
  1. Select a search command (search, tstats, stats)
  2. Choose your index and sourcetype
  3. Set time range or use custom time
  4. Add filters using the filter section
  5. Click "Generate SPL Command"
  6. Copy and paste into Splunk
SPL Operators
  • = : Equal to
  • != : Not equal to
  • > : Greater than
  • < : Less than
  • IN : Value in list
  • MATCHES : Pattern matching
